# Cross PR Roadmap

This public roadmap describes the product direction at a high level only.

## Current Focus

- describe Cross PR as a Telegram mutual promotion product
- make the public materials useful for search discovery
- explain the audience and use cases without exposing private implementation
- keep the repository safe as a non-runnable public overview

## Product Direction

- Telegram partner discovery
- channel and bot profile concepts
- mutual PR bundle planning
- structured outreach workflows
- lightweight campaign status concepts
- clearer public positioning for SEO

## Not Included Publicly

- production source code
- private matching logic
- internal prompts
- analytics exports
- deployment instructions
- credentials or API configuration
