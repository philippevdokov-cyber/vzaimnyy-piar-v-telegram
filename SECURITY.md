# Security Policy

This public repository is documentation-only.

It does not contain production code, secrets, API keys, bot tokens, private prompts, deployment files, databases, analytics exports, matching algorithms, or personal data.

## Reporting

If you believe this repository accidentally exposes sensitive information, open a GitHub issue with a short description of the file and line. Do not include secrets in the issue text.

## Public Boundary

The public materials are intentionally high-level. They are meant for product discovery and SEO, not for cloning or operating the real Cross PR system.
